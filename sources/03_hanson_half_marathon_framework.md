# Hanson Half-Marathon Framework

## Default Framework

Use Hanson / Hansons Method principles as the default framework for the current cycle, adapted to the half marathon rather than copied from Advanced Hansons Marathon Method.

## Emphasis

- Cumulative fatigue
- Moderate long runs
- Consistent frequency
- Threshold, stamina, and half-marathon-pace work
- Easy days kept genuinely easy

## SOS Structure

The plan may use structured SOS days, but load should be earned through consistency and recovery signals.
